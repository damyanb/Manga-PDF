# Manga PDF

Script de pyhton que toma pantallazos del navegador y lee los subtitulos, cuando los dialogos cambian, toma un nuevo screenshots para despues crear un PDF con todos los screenshots, la idea es convertir un capitulo de una serie en un manga para porder leerlo
